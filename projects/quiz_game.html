Quiz Game

Spring 2021 (~1 month)

Overview

The Ultimate Quiz is a comprehensive quiz game that utilizes web scraping, with BeautifulSoup, to extract information from online articles, to then generate of quiz-based questions to present to the user. In addition, a custom-made mastery system, drawing inspiration from the Leitner System and SuperMemo algorithms, was implemented to optimize and track player progress throughout the gameplay. This mastery system would implemented to ensure the player is always learning, similar to quizlet.The game also features a dedicated stats screen, which displays player progress and performance data.
